# In The Moment
Create Moments by recording challenges with unique, unexpected sounds.
The app works best on Android and Chromium web browsers. There are currently issues with Safari that I'm still trying to fix. This app was made with Codex and GPT 5.6 Terra with various sound effects from freesound.org and Adobe Firefly generation. Codex was used for the source code in its entirety and pushing commits to GitHub. No code was manually editied for this web app. The entire experience runs in browser. GPT 5.6 was also used to generate a list of challenges with descriptions and ideas for what sound effect should be used.
