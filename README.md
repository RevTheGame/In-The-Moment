# In The Moment

**Create memorable moments by completing spontaneous real-world challenges with a unique and unexpected sound effect.**

🔗 **Live Demo:** https://revthegame.github.io/In-The-Moment/

## Overview

In The Moment encourages people to capture everyday experiences through simple, open-ended challenges.

The entire experience runs in the browser.

---

## Setup

No installation is required.

https://revthegame.github.io/In-The-Moment/

**Recommended browsers**

* Chrome
* Edge
* Brave
* Opera

**Known Issue**
Safari currently has compatibility issues that are still being investigated.

# AI Development Process

This project was built with Codex and GPT 5.6.

(and most of this README)

* Building the complete browser application
* Implementing challenge selection logic
* Integrating camera recording
* Creating the UI and interaction flow
* Handling browser compatibility improvements
* Debugging issues through iterative conversations
* Writing and refactoring JavaScript, HTML, and CSS
* Creating Git commits and pushing updates to GitHub throughout development

**No source code was manually edited.** Every code change originated from prompts to Codex.

---

## How GPT-5.6 Was Used

GPT-5.6 Terra served as the creative design partner.

It was used to:

* Brainstorm challenge ideas
* Expand challenge descriptions
* Recommend unexpected and comedic sound effects
* Refine gameplay ideas

---

## Key Technical Decisions

Several design decisions were made with the assistance of GPT-5.6 and implemented by Codex:

* Keep the entire application browser-based with no installation.
* Use vague, open-ended challenges so users can interpret them creatively.
* Store challenge content directly within the application instead of relying on external services.
* Prioritize mobile usability while remaining functional on desktop browsers.
* Keep the architecture lightweight and dependency-free for easy deployment through GitHub Pages.
